### 介绍
cli工具 运行命令行自动生成react或者vue脚手架

### 使用
* 全局安装 `npm i adomin-cli -g`
* 生成react脚手架 `adomin-cli init react projectName`
* 生成vue脚手架 `adomin-cli init vue projectName`

### Description
